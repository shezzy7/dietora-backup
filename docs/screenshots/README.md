# DIETORA — Screenshots Guide

This folder contains screenshots of all DIETORA pages for FYP documentation.

## How to Take Screenshots

Run the project (see main README), then capture each page below.
Recommended tool: Windows Snipping Tool (Win+Shift+S) or browser screenshot extension.
Recommended resolution: 1440x900 (desktop) and 390x844 (mobile/iPhone 14).

## Screenshot Tips

1. **Landing page**: Scroll to show hero section first, take 2-3 screenshots covering all sections
2. **Dashboard**: Make sure to have a meal plan generated so stats show real data
3. **Dark mode**: Toggle dark mode in TopBar before screenshotting
4. **Mobile**: Use Chrome DevTools (F12) → Toggle device toolbar → iPhone 14 preset
5. **Chatbot**: Click the 🤖 button bottom-right to open before screenshotting
